# Payment API v1 Deprecation Notice

## Overview

Payment API v1 has been deprecated and will be removed in the future.

## Timeline

- v2 Release Date: May 7, 2026
- Deprecation Start Date: June 1, 2026
- v1 Shutdown Date: December 1, 2026

## Deprecated Endpoint

POST /api/v1/payments

## New Endpoint

POST /api/v2/payments

## Improvements in v2

- Multi-currency support
- Better security with tokenization
- Transaction tracking
- Improved error handling

## Migration Guide

Please migrate your application to v2 before the sunset date.
