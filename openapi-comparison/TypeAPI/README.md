# TypeAPI Demo

This folder contains a simplified type-first API description for the Library Management API.

## Purpose
This format is included for comparison with OpenAPI, API Blueprint, RAML, and TypeSpec.

## Notes
- TypeAPI here is a custom typed description style
- It is useful for explaining data models and endpoint contracts in a simple way
- Unlike OpenAPI, it does not directly integrate with major tooling ecosystems

## Run / Usage
This format is documentation-only in this demo.
It is intended for comparison, not for direct code generation.
